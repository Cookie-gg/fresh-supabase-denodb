# fresh project

Trying to make a blog system with Fresh, Supabase and Denodb.

> ref: https://zenn.dev/azukiazusa/articles/fresh-tutorial

### Usage

Create env file:

```
DB_USER=""
POSTGRES_DB=""
DB_PORT=0000
DB_PASSWORD=""
DB_HOST=""
```

Start the project:

```
deno task start
```

This will watch the project directory and restart as necessary.
